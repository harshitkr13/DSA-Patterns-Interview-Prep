# DSA Pattern Wise Solutions (C++)

This repository contains my **pattern-wise revision of Data Structures & Algorithms** using **C++**.

The goal of this project is to understand **how to recognize algorithmic patterns used in coding interviews**, rather than memorizing isolated solutions.

I am documenting my learning journey while revising DSA from the **A2Z DSA roadmap by Striver**, and organizing problems based on commonly used **problem solving patterns**.

---

## Repository Goal

• Strengthen **DSA fundamentals**

• Learn to **identify patterns in interview problems**

• Maintain a **structured collection of solutions**

• Document **clean and well-commented C++ implementations**

• Build a **knowledge base for coding interview preparation**

---

## Patterns Covered

This repository is organized **pattern-wise** to make learning more structured.

| Pattern | Description |
|------|------|
| Binary Search | Searching and optimization problems using divide and conquer |
| Two Pointers | Efficient array/string traversal techniques |
| Sliding Window | Optimized subarray and substring problems |
| Prefix Sum | Range queries and cumulative computations |
| Stack / Monotonic Stack | Next greater element and similar problems |
| Heap / Priority Queue | Top K and scheduling problems |
| Graph (BFS/DFS) | Graph traversal and connectivity problems |
| Backtracking | Recursive exploration of solution spaces |
| Dynamic Programming | Optimization problems with overlapping subproblems |

---

Each pattern folder contains:

• Concept explanation  
• How to recognize the pattern  
• Interesting / must try interview questions  
• C++ implementations with comments  
• Corresponding LeetCode problem links
